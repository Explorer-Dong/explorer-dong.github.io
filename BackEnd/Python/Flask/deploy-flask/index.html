部署 Flask

Flask 是使用 Python 作为后端语言的 Web 微框，旨在帮助开发者快速开发出一个小型网站。个人示例项目就采用了 Flask 框架，前端直接使用 Flask 自带的 Jinja2 进行渲染，数据库采用 MySQL。项目地址 YunJinWeb，下面为以此项目为 demo 进行云服务器部署的过程

（四）终止或重启 uwsgi 服务器

如果想要终止 uwsgi 服务器，逻辑就是找到该服务器的所有进程并全部终止即可。如果因为项目更新等原因需要重启应用，需要先终止 uwsgi 服务器后再重新启动 uwsgi 服务器。
